Acerca de mí

¡Hola! Mi nombre es Kevin Julián Guerrero Penagos. Soy estudiante de la Universidad Cooperativa de Colombia, donde estudio desde el año 2022, nací el 15 Diciembre del año 2003 y actualmente curso el séptimo semestre.

Me apasiona aprender sobre ciencia de datos, analítica de datos, y el diseño tanto frontend como backend. Crear y diseñar páginas web se ha convertido en una práctica que disfruto mucho y me reta constantemente a crecer profesionalmente.

Además, me considero una persona proactiva en el trabajo en equipo. He tenido la oportunidad de liderar proyectos, pero también sé desempeñarme como un miembro que apoya a mis compañeros y al líder para alcanzar los objetivos propuestos.

Proyectos

Proyecto Pijao Del Sol (Hotel)

Este proyecto consistió en la creación de un sistema de gestión para un hotel, donde se manejan diversas funcionalidades como la reserva de habitaciones, registro de clientes, gestión de pagos y control de disponibilidad de las habitaciones. Se implementó utilizando tecnologías web modernas y bases de datos para garantizar un sistema eficiente y fácil de usar para los empleados y clientes del hotel.

Proyecto Login con Docker

Este proyecto tiene como objetivo la implementación de un sistema de autenticación de usuarios utilizando Docker. Se creó un contenedor Docker para facilitar la implementación y el despliegue del sistema de login en diferentes entornos. El sistema permite a los usuarios registrarse e iniciar sesión de manera segura, utilizando buenas prácticas de desarrollo y contenedores para asegurar la portabilidad del proyecto.
